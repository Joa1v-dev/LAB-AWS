<h1> Relatório Sobre o Aprendizado da AWS até o momento.</h1>

Atualmente a AWS é referência no mercado de computação em nuvem, a plataforma da amazon conta com inúmeros serviços para auxiliar o usuário, banco de dados, segurança, servidores virtuais e entre outros. Em vez de comprar um servidor físico, você aluga recursos pela internet, para utilizar da forma que desejar.
